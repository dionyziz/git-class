Προετοιμασία για συμμετοχή

  1. Φέρτε λάπτοπ!

  2. Δημιουργήστε λογαριασμό στο GitHub.

  3. Παρακαλούμε κατεβάστε και εγκαταστήστε το git στον υπολογιστή σας πριν το σεμινάριο: https://git-scm.com/downloads

    • Για Windows: Κατεβάστε και εγκαταστήστε το binary από την παραπάνω διεύθυνση.
    • Για Mac: Τρέξτε την εντολή git και ακολουθήστε τις οδηγίες.
    • Για Linux: Εγκαταστήστε το με τον package manager σας, π.χ. για Debian/Ubuntu χρησιμοποιήστε την εντολή sudo apt-get install git.

    Περισσότερες οδηγίες για την εγκατάσταση: https://git-scm.com/book/…/v2/Getting-Started-Installing-Git

    Επειδή το bandwidth μας κατά τη διάρκεια του σεμιναρίου θα είναι περιορισμένο, σας παρακαλούμε να το κάνετε από πριν!

  4. Για μια πρώτη επαφή κι εξοικείωση με το git πριν ξεκινήσουμε, μπορείτε να αφιερώσετε 15 λεπτά εδώ.
